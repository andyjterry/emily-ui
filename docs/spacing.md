# Spacing

## Purpose
Show how spacing scale values power padding, margin, gap, and related utilities.

## Minimal Example
```json
{
  "spacing": {
    "scale": {
      "4": "1rem",
      "8": "2rem"
    }
  }
}
```

```html
<section class="p-4 md:p-8"></section>
```
