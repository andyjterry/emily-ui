# Configuration

## Purpose
Explain how `emily.config.json` controls tokens, output files, purge settings, and feature flags.

## Minimal Example
```json
{
  "output": {
    "css": "dist/emily.min.css",
    "fullCss": "dist/emily.css"
  },
  "manifest": true,
  "intellisense": true
}
```
