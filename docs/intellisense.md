# IntelliSense

## Purpose
Explain how to generate IntelliSense metadata JSON for editor and tooling integrations.

## Minimal Example
```json
{
  "intellisense": {
    "enabled": true,
    "output": "dist/emily.intellisense.json"
  }
}
```

```bash
npx emily-css build
```
