# Colours

## Purpose
Describe brand colour tokens and how EmilyCSS generates utility classes from palette values.

## Minimal Example
```json
{
  "colours": {
    "brand": "#0077B6"
  }
}
```

```html
<div class="bg-brand-80 text-white"></div>
```
